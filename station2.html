Hot keys are:

Play/pause => Ctrl + /

Rewind => Ctrl + <

Fast_forward => Ctr + >

Stop => Ctr + s

Insert timecode => Ctr + i

Click on the relative text box so as to edit the time code and click play(ctrl + /) button to resume the audio.

Fast forward/backward will fast forward and backward 3 seconds respectively.

After filling up all the timestamp press "I'm Done!" button to submit your task.
